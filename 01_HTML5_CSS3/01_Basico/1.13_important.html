!important no CSS

O que é?

!important força uma regra CSS a ser aplicada, mesmo se houver outras regras mais específicas.

Como usar?

Coloque !important depois do valor:

color: red !important;

Exemplo:

Este texto deveria ser azul, mas fica vermelho por causa do !important

Este texto tem ID (verde) mas fica roxo por causa do !important

Quando usar?

  • Quando outra regra está "ganhando"
  • Para sobrescrever estilos externos
  • Em casos de emergência

Cuidado!

Use pouco! O !important pode tornar o CSS difícil de manter.
