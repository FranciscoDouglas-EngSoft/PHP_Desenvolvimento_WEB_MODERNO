Float no CSS

O que é?

Float faz um elemento "flutuar" para a esquerda ou direita, permitindo que outros elementos contornem ao redor dele.

Valores:

  • left - Flutua para a esquerda
  • right - Flutua para a direita
  • none - Sem float (padrão)

Este texto contorna a caixa azul que está flutuando à esquerda. Veja como o texto se ajusta ao redor da caixa.

Exemplo - Float Right:

Float Right

Agora a caixa flutua à direita e o texto se ajusta contornando ela do lado esquerdo.

Uso comum - Imagem com texto:

IMG

Este é um uso muito comum do float: colocar uma imagem flutuando à esquerda e o texto contornando ela, como em jornais e revistas.

O texto continua fluindo naturalmente ao redor da imagem.

Clear:

Use clear: both para parar o efeito do float e fazer o próximo elemento aparecer abaixo.

Importante:

⚠️ Float é antigo! Hoje usamos Flexbox ou Grid para layouts. Float é usado principalmente para texto contornar imagens.
