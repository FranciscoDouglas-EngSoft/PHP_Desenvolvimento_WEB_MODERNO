Propriedades de Textos em CSS3

O CSS3 oferece diversas propriedades para estilizar textos, permitindo ajustar cores, fontes, tamanhos, espaçamentos e muito mais.

1. Color (cor do texto)

CSS:
color: #E91E63;
Resultado:

Este texto está com a cor personalizada.

2. Font-family (família de fonte)

CSS:
font-family: Arial, sans-serif;
Resultado:

Este texto usa fonte Arial (sem serifa).

Este texto usa Times New Roman (com serifa).

Este texto usa fonte monospace.

3. Font-size (tamanho da fonte)

CSS:
font-size: 12px;
font-size: 1.5em;
font-size: 150%;
Resultado:

Este texto tem tamanho de 12px.

Este texto tem tamanho de 1.5em.

Este texto tem tamanho de 150%.

4. Font-weight (peso/espessura da fonte)

CSS:
font-weight: normal;
font-weight: bold;
font-weight: 900;
Resultado:

Este texto tem peso normal.

Este texto tem peso negrito (bold).

Este texto tem peso 900 (extra-bold).

5. Text-align (alinhamento de texto)

CSS:
text-align: left;
text-align: center;
text-align: right;
text-align: justify;
Resultado:

Este texto está alinhado à esquerda.

Este texto está alinhado ao centro.

Este texto está alinhado à direita.

Este texto está justificado. O alinhamento justificado faz com que o texto se alinhe tanto à margem esquerda quanto à margem direita, criando linhas de tamanhos iguais, exceto possivelmente a última linha.

6. Text-decoration (decoração de texto)

CSS:

text-decoration: overline;
text-decoration: line-through;
text-decoration: none;
Resultado:

Este texto tem sublinhado.

Este texto tem linha acima.

Este texto tem uma linha no meio (riscado).

Este link não tem decoração (none)

7. Text-transform (transformação de texto)

CSS:
text-transform: uppercase;
text-transform: lowercase;
text-transform: capitalize;
Resultado:

Este texto está em maiúsculas.

Este TEXTO ESTÁ em minúsculas.

este texto tem a primeira letra de cada palavra em maiúscula.

8. Line-height (altura da linha)

CSS:
line-height: 1.2;
line-height: 2.0;
Resultado:

Este texto tem altura de linha padrão (1.2). Esta propriedade controla o espaçamento entre as linhas de texto, o que pode melhorar a legibilidade.

Este texto tem altura de linha maior (2.0). Observe como há mais espaço entre as linhas, tornando o texto mais arejado e fácil de ler em alguns casos.

9. Letter-spacing (espaçamento entre letras)

CSS:

letter-spacing: 3px;
Resultado:

Este texto tem espaçamento entre letras de 1px.

Este texto tem espaçamento entre letras de 3px.

10. Word-spacing (espaçamento entre palavras)

Resultado:

Este texto tem espaçamento entre palavras de 2px.

Este texto tem espaçamento entre palavras de 10px.

11. Text-shadow (sombra de texto)

12. White-space (tratamento de espaços em branco)

CSS:
white-space: normal;
white-space: nowrap;
white-space: pre;
Resultado:

Este texto quebra normalmente quando atinge o limite da caixa.

Este texto não quebra mesmo que atinja o limite da caixa (nowrap).

Este texto mantém todos os espaços e quebras de linha.

13. Text-overflow (overflow de texto)

overflow: hidden;

width: 150px;

Este é um texto longo que será cortado e mostrará reticências (...) no final.

Combinando Propriedades

Exemplo de estilo tipográfico completo:
p {
  color: #333;
  font-family: 'Georgia', serif;
  font-size: 1.1em;
  line-height: 1.6;
  text-align: justify;
  letter-spacing: 0.5px;
  word-spacing: 1px;
}

Nota: As propriedades CSS para texto são uma parte fundamental do design web, permitindo criar layouts com boa legibilidade e expressão visual. É importante escolher combinações de propriedades que mantenham o texto acessível e fácil de ler.
